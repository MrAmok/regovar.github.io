Contribuer

Regovar est un projet libre et ouvert, où tout le monde peut participer. Peu importe vos compétences, toute aide est la bienvenue. Pour vous donner une idée, voici ci-dessous une liste non exhaustive de l'aide que vous pouvez apporter.

  • Tester l'application
  • Nous dire ce que vous en pensez
  • Proposer vos idées
  • Indiquer les bugs ou difficultés que vous rencontrez via notre gestionnaire de tickets
  • Améliorer les traductions (français / anglais)
  • Participer au développement
  • Partager vos pipelines
  • Aider à maintenir la documentation ou le site web à jour…

Si vous êtes intéressé, voici une rapide présentation des principaux outils que nous utilisons pour travailler :

  • GitHub : pour le partage et le versionnage du code source ainsi que la gestion des tickets ;
  • ReadTheDoc : pour gérer la documentation en ligne du projet ;
  • Téléphone, mail ou discord : pour discuter.
